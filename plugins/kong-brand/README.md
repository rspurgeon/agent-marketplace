# Kong Brand Plugin

Encodes the Kong Brand Guidelines 2026 v1.1 so Claude Code can generate and review on-brand content.

## Skills

### `/kong-brand:brand-review`

Apply Kong brand guidelines when generating or modifying code, CSS, HTML, web pages, demos, or any visual content.

**Example usage:**

```
/kong-brand:brand-review build a landing page for Kong AI Gateway
/kong-brand:brand-review update the colors in this CSS to match Kong brand
/kong-brand:brand-review review this HTML for brand compliance
```

## What's Included

- Complete Kong color palette (Dark Green, Electric Lime, Bay, White, 10-step gray scale)
- Typography specs (Funnel Sans, Roboto Mono, Space Grotesk) with CSS imports
- Logo usage rules and theme variants
- Official product name list
- Graphic style guidance (blueprint grids, dark-dominant aesthetic)
- CSS custom property definitions ready to use

## Source

Based on Kong Brand Guidelines 2026 v1.1. For branding questions contact design@konghq.com.
